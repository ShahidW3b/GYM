## 🚀 Overview

C4 ProGYM is a modern and responsive gym management platform that provides role-based access for different users.  
It focuses on delivering a smooth and intuitive experience for managing fitness activities, memberships, and interactions.

---

## 👥 User Roles

- 👤 **Users** – Access gym services and personal dashboard  
- 🧑‍🏫 **Trainers** – Manage training sessions and clients  
- 🛠️ **Admins** – Oversee system operations and user management  

---

## ✨ Features

- Fully responsive design (mobile-first)  
- Multi-role authentication system  
- Clean and modern UI/UX  
- Fast and lightweight performance  
- Interactive and user-friendly interface  

---

## 🛠️ Technologies Used

- **HTML5** – structure and layout  
- **CSS3** – styling and responsiveness  
- **JavaScript** – dynamic functionality  



---

## 🎯 Purpose

The goal of this project is to demonstrate a complete front-end gym management system with role-based interaction and a modern user interface.
